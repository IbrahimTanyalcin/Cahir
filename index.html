Cahir
Proxies, tagged-templates and webcomponents, side by side
In about 160 lines of code

Cahir (`ch`) is a function proxy that can be interchangibly used for method chaining and tagged templates. You can:
  • reuse tagged templates via ch.pickle`...`
  • label variables like `JSON` ch`method variable:${some_value}`
  • spread the values within tagged templates ch`method ...${[some_array]}`
  • or do both of above
  • return webcomponents ch`<game-card ${{ data: {values, d} }}/>`
